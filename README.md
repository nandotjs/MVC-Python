# MVC pattern in Python

- Model: data
- View: UI
- Controller: logic

## Setup

- Use virtual environment
- Use Pylint and pre-commit
- Use SQLAlchemy to connect SQLite

## Structure

1. Make models
    - entities
    - repositories 
    - interfaces

2. Make controllers
   - test
   - interfaces

3. Make Views

4. Make Main
   - server config
   - routes
   - composers
